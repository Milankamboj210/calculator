# 🧮 Modern Calculator

A sleek and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs all basic arithmetic operations with a modern UI, smooth animations, and keyboard support.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (AC)
- ⌫ Backspace
- 📐 Parentheses Support
- 🔢 Decimal Calculations
- ⌨️ Keyboard Support
- 📱 Fully Responsive Design
- 🎨 Modern Dark Theme UI
- ⚡ Smooth Hover & Click Animations
- ❌ Error Handling for Invalid Expressions
